## 코드
본 코드들은 프로젝트를 진행하며 결과물을 도출할 때 사용된 코드로, `직접 수집 DATA, 전처리 DATA`를 이용해 구동 가능합니다. 
ESDA를 제외한 모든 코드는 colab 환경에서 드라이브 마운트를 통해 진행하였습니다. 

```kotlin
1) folium
*folium 분석은 환경 설정 후, 런타임을 종료한 후 다시 시작해야 그래프에 있는 폰트가 깨지거나, 이미지 파일 저장에 오류가 없습니다. 
    (1) 서울시사회복지시설현황.ipynb.ipynb
        - 연결 데이터 : 사회복지시설_시각화용_최종.xlsx
    (2) 수급자총계_시각화용.ipynb, 주민등록인구_추가분석.ipynb
        - 연결 데이터 : 한부모수급자_시각화용.xlsx, test2.xlsx
    (3) 지역별총예산투입.ipynb
        - 연결 데이터 : 키워드_확장버전_예산.xlsx, 하위 5개.xlsx, 상위 5개.xlsx
2) ESDA
*ESDA 파일들은 파이썬이 아닌, 'R'을 기반으로 구동됩니다.
따라서, 활용 데이터 형식이 dbf 확장자로 되어있기에 ESDA는 예외적으로 활용 데이터까지 함께 첨부해두었습니다. 
    (1) ESDA 서울_저소득한부모 : 연결 데이터 동일 파일내 존재
    (2) ESDA 시설_파생변수 : 연결 데이터 동일 파일내 존재
    (3) ESDA 예산_파생변수 : 연결 데이터 동일 파일내 존재

3) 예산 전처리 코드
예산을 키워드를 중심으로 도출해냈기에 전처리 시 파이썬을 활용하여 해당 코드를 같이 첨부해두었습니다.
    - 키워드예산 천저리.ipynb
        - 연결 데이터 : 서울시_법정동코드.csv, 서울지자체 총예산 폴더 내 지역별 csv 파일

4) 워드클라우드_하트버전.ipynb
        - 연결 데이터 : 총_민원_데이터.xlsx

5) 실루엣 계수 및 K-means.ipynb
    - 연결 데이터 : pca_f.xlsx
    
6) Lasso 회귀분석.ipynb
    - 연결 데이터 : 정규화_파생변수.xlsx

7) PCA.ipynb
    - 연결 데이터 : 촤종_Eng.xlsx
```
